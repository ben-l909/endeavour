<h1>What is this</h1>
<p>
  Endeavour is a binary analysis toolkit that combines the power of LLMs with traditional methods (MBA simplification, IR lifting) to help reverse engineers understand obfuscated code.
  <br /><br />
  This project is <b>more than that</b>, though - it's a proof of concept showing that LLMs can replace much of a modern corporation’s workflow, from QA testing and product design to engineering and releases.
  <br /><br />
  I haven't manually written a single line of code for this project, despite spending most of my life doing exactly that. Part of me is sad that the passion that used to go into the tools we use day to day is fading, as corporations and individuals move toward writing instructions instead of code.
  <br /><br />
  Another part of me is excited to see what I can build now that constraints on complexity have shifted. Time is less of a bottleneck, and projects that once had year-long deadlines can now be completed in weeks or months. I don't think AI will be the end of software engineers, more like a new beginning.
  <br /><br />
  If you'd like to give this a try, I'll let the agents explain how it works below.
</p>
